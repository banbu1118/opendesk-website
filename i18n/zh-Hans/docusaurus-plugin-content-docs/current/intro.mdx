<h2 style={{ marginTop: '0' }}>✨ 产品简介</h2>

<Card>
OpenDesk 是一个基于 **Proxmox VE（PVE）** 构建的开源 VDI 管理平台。

通过统一的 Web 控制台，实现虚拟桌面生命周期管理、资源调度与访问控制。

用户通过 **RDP + RDPGW 网关** 访问桌面，实现安全、高效、可扩展的桌面云体验。
</Card>
