# 关于

> 一个面向未来的 VDI（虚拟桌面基础设施）开源管理平台

---

## 🌱 我们在做什么

OpenDesk 致力于构建一个**简单、高效、开放的虚拟桌面管理平台**，让企业和个人能够更轻松地部署、管理和使用虚拟桌面环境。

我们基于 Proxmox VE（PVE）生态，通过统一的 Web 控制台，将虚拟机资源调度、桌面分发与远程访问整合在一起，让 VDI 不再复杂。

---

## 🎯 我们的目标

我们相信：

- VDI 不应该只属于大型企业
- 虚拟化不应该复杂且封闭
- 每个人都应该能轻松拥有自己的云桌面平台

OpenDesk 的目标是：

- 降低 VDI 的使用门槛
- 提供可扩展的开源架构
- 构建一个开放的虚拟桌面生态
