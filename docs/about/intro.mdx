# About

> A future-oriented open source VDI (Virtual Desktop Infrastructure) management platform

---

## 🌱 What We're Doing

OpenDesk is dedicated to building a **simple, efficient, and open virtual desktop management platform** that enables businesses and individuals to deploy, manage, and use virtual desktop environments more easily.

Based on the Proxmox VE (PVE) ecosystem, we integrate virtual machine resource scheduling, desktop distribution, and remote access through a unified web console, making VDI no longer complex.

---

## 🎯 Our Goals

We believe:

- VDI should not be exclusive to large enterprises
- Virtualization should not be complex and closed
- Everyone should be able to easily have their own cloud desktop platform

OpenDesk's goals are:

- Lower the threshold for using VDI
- Provide a scalable open source architecture
- Build an open virtual desktop ecosystem
