<h2 style={{ marginTop: '0' }}>✨ Product Introduction</h2>

<Card>
OpenDesk is an open source VDI management platform built on <strong>Proxmox VE (PVE)</strong>.

Through a unified web console, it enables virtual desktop lifecycle management, resource scheduling, and access control.

Users access desktops through <strong>RDP + RDPGW</strong> for a secure, efficient, and scalable desktop cloud experience.
</Card>

---

## ⚡ Core Capabilities

<a id="features"></a>

<CardGrid>

<Feature title="🖥️ Virtual Machine Management" desc="Unified VM lifecycle management based on PVE API, supporting creation / start/stop / allocation / recycling" />

<Feature title="🔐 Secure Access" desc="Unified RDPGW entry with session isolation, permission control, and access auditing" />

<Feature title="⚡ High Performance Scheduling" desc="Low-latency connection experience with GPU acceleration and dynamic resource allocation" />

<Feature title="📊 Operational Visualization" desc="VM status monitoring, user session management, resource usage statistics" />

</CardGrid>

<a id="architecture"></a>

<CardGrid>
